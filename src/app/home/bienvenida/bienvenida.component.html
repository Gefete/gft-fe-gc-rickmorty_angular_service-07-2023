Bienvenido

Este es un projecto Angular que lleva a cabo una recopilación de datos de una API de Rick y Morty
En esta versión del proyecto practicamos las llamadas de datos desde una API, en el ejercicio
realizado muestra 5 tarjetas aleatorias de la API en el que si seleccionamos una podremos ver más
detalles del personaje.
